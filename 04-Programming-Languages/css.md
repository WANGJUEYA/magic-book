## 元素选择器


+ 基本选择器:
    - 标签选择器：统一定义常用标签的基本样式 $(‘body’)
    - 类选择器：“class”，以.开头后面接自己定义的类名 $(‘.class)
    - ID选择器：“id”，以#开头后面接名字 $(‘#id’)
    - 通配选择器：用*表示，可以可以一次性定义所有元素

+ 高级选择器:
    - 后代选择器：用空格隔开
    - 交集选择器：用.隔开
    - 并集选择器：用，隔开
