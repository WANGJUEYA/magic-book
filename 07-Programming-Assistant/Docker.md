## 常用命令

### 查看启动的任务

```shell
docker ps
```

### 查看所有任务

```shell
docker ps -a
```

### 启动/停止任务

```shell
docker start $contaniner-id
docker stop $contaniner-id
```

### 其他常用命令

```shell
docker -v # 查看当前版本
```
